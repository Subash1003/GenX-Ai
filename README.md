# GenX-AI 

An AI-powered chat assistant built with React + Vite, powered by Google Gemini.

## Getting Started

1. Clone the repo
```bash
   git clone https://github.com/subash1003/GenX-Ai.git
```

2. Install dependencies
```bash
   npm install
```

3. Create a `.env` file and add your Gemini API key

4. Run the app
```bash
   npm run dev
```
